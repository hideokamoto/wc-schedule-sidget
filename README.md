## Installation ##

1. Upload `wc-schedule-widget.zip` to the `/wp-content/plugins/` directory
2. Unzip uploaded file.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Place `WordCamp Scheduler Widget` in your site.
